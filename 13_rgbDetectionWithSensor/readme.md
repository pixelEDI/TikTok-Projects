# General info

In this small project, we detect RGB colors using Adafruit's TCS34725 RGB sensor and display them on a WS2812B LED ring.

## Hardware
- For this project we used an Arduino Uno.
- TCS34725 RGB Sensor
- LED Ring 5V RGB WS2812B
